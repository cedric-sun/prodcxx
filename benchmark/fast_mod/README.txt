Starting from -O2, G++ inline both `mod` and `fast_mod` call anyway,
regardless if you give the keyword `inline` for "hint".

Run the following to check.

        g++ -O2 -S static_inline_bm.cc
        g++ -O2 -S just_static.cc
        diff static_inline_bm.s just_static.s


clang++ 10.0.1
g++ 10.2.0

Clang++ generated executable is always slower in this test then that of g++.
